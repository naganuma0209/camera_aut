概要
このアプリは、Webカメラを使って写真を撮影し、撮影日時・PC名とともにCSVファイルへ自動記録するツールです。
Tkinterで構築されたシンプルなGUIを使って、ワンクリックで撮影ができます。
プログラムは教育・業務・記録用途など、汎用的に利用できます。

機能
✅ リアルタイムプレビュー：カメラ映像を常時表示
📝 ファイル名入力機能：テキストボックスで任意のファイル名を指定可能
💾 自動保存：撮影した画像を photos フォルダに自動保存
📊 撮影ログ：photo\_log.csv に「日時・ファイル名・PC名」を記録
🎨 視覚フィードバック：撮影ボタンが撮影成功時に一時的に緑色に変化
🧩 GUI操作のみで完結：「撮影」「終了」ボタンで操作可能

動作環境
項目	推奨環境
Python	3.8 以上
ライブラリ	opencv-python, tkinter（標準）, csv, socket, datetime, os
対応OS	Windows / macOS / Linux（カメラ対応環境）
インストールコマンド
pip install opencv-python


使い方
プログラムを実行
python camera\_app.py
ウィンドウが表示される
「ファイル名」欄に任意の名前を入力（未入力なら photo\_YYYYMMDD\_HHMMSS.jpg）
「撮影」ボタンを押すと写真を撮影・保存
ボタンが一時的に緑色になり、撮影成功を示します

終了方法
「終了」ボタンを押すとプログラムを終了します

保存内容
📁 保存フォルダ構造
camera\_app.py
photo\_log.csv        ← 撮影履歴
photos/              ← 撮影した画像
&nbsp; ├── photo\_20251103\_210012.jpg
&nbsp; ├── sample\_20251103\_210045.jpg


🧾 撮影ログ例 (photo\_log.csv)
日時	ファイル名	PC名
2025-11-03 21:00:12	photos/test\_20251103\_210012.jpg	Gnu-PC
2025-11-03 21:00:45	photos/sample\_20251103\_210045.jpg	Gnu-PC
コード構成（概要）

関数	役割
capture\_photo()	写真撮影・保存・ログ追記・ボタン演出
show\_frame()	カメラ映像をリアルタイムに表示
close\_app()	カメラ解放とウィンドウ終了処理
root.after()	GUIの定期更新（映像表示・ボタン色戻し）

注意事項
カメラが接続されていない場合、映像が表示されません。
カメラが他のアプリで使用中の場合、エラーが発生する場合があります。
cv2.imshow() のプレビューウィンドウを手動で閉じず、「終了」ボタンで閉じてください。
photos フォルダと photo\_log.csv は自動で生成されます。
